Easy Password Generating

AirPassGen provides a quick and easy way to create sets of unique hash codes. Whether you need a random password or a set of unique invitation codes, AirPassGen lets you create large numbers of hashes easily.

The software runs on Adobe’s AIR runtime and can be used on Apple OS X and Microsoft Windows.
